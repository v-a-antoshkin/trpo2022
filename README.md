### Язык разметки MarkDown

**Параграф**  
автоматически __переносится__ на ~~следующую~~ строку
*Параграф 2*

_Параграф 3_

#### Нумерованный список

1. Первый пункт.
   1. Подпункт 1.
   1. Подпункт 2.
1. Третия пункт.

   Вложенный текст
1. Второй пункт.

#### Маркерованный список

* Первый пункт
   * Вложенный подпункт
      * Вложенный во вложенный подпункт
         * Вложенный во вложенный подпункт
            * Вложенный во вложенный подпункт
- Второй пункт
+ Третий пункт

```c#
   MyClass myClass = new();
```
